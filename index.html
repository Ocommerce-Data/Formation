Aggregating Data with Window Functions

SUM(), AVG(), MIN(), MAX(): Calculate the running total of sales for each employee.


SELECT 
    sale_id, 
    employee_id, 
    SUM(quantity * price) OVER (PARTITION BY employee_id ORDER BY sale_date) AS running_total
FROM 
    sales;

Partitioning and Ordering

PARTITION BY Clause: Calculate total sales for each region.


SELECT 
    sale_id, 
    region, 
    SUM(quantity * price) OVER (PARTITION BY region) AS region_total_sales
FROM 
    sales;

ORDER BY Clause: Calculate the cumulative sales for each region, ordered by sale_date.


SELECT 
    sale_id, 
    region, 
    SUM(quantity * price) OVER (PARTITION BY region ORDER BY sale_date) AS cumulative_sales
FROM 
    sales;

Advanced Window Functions

LAG() and LEAD(): Compare current sale quantity with the previous sale quantity for each product.


SELECT 
    sale_id, 
    product_id, 
    quantity, 
    LAG(quantity) OVER (PARTITION BY product_id ORDER BY sale_date) AS prev_quantity
FROM 
    sales;

FIRST_VALUE() and LAST_VALUE(): Retrieve the first and last sale price for each product within each region.


SELECT 
    sale_id, 
    product_id, 
    region, 
    FIRST_VALUE(price) OVER (PARTITION BY product_id, region ORDER BY sale_date) AS first_price, 
    LAST_VALUE(price) OVER (PARTITION BY product_id, region ORDER BY sale_date ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING) AS last_price
FROM 
    sales;
